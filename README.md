# 🚦 Traffic Violation Detection using YOLOv8

This project detects traffic violations using computer vision and YOLOv8.

## Features
- 🚨 Triple riding detection
- 🪖 No helmet detection
- 🚦 Red light violation (cars & bikes)
- 📸 Number plate cropping (future OCR ready)

## Tech Stack
- Python
- YOLOv8 (Ultralytics)
- OpenCV

## How it works
1. Input image/video
2. YOLO detects vehicles & signals
3. Rule-based violation logic applied
4. Evidence saved automatically

## Future Improvements
- Helmet-specific YOLO model
- Number plate OCR
- Video-based real-time detection
